Mengapa Perlu Logika?

Komputer hanya bisa mengambil keputusan berdasarkan true atau false. Operator logika membantu kita menterjemahkan masalah ke bentuk pertanyaan yang bisa dijawab komputer.

Validasi

Memastikan input pengguna sesuai syarat sebelum diproses.

Percabangan

Menentukan cabang if/else yang benar dieksekusi.

Automasi

Menggabungkan beberapa kondisi untuk mengontrol alur kerja otomatis.

Operator Perbandingan

Simbol & Makna

  • == — sama dengan
  • != — tidak sama
  • >, < — lebih besar / kecil
  • >=, <= — lebih besar/kecil atau sama

Contoh Cepat

Semua operator menghasilkan nilai bool:

  • 5 == 5 ➜ True
  • 7 != 3 ➜ True
  • 10 <= 9 ➜ False
  • 'AI' == 'ai' ➜ False

Boolean & Truthiness

Nilai Boolean

Python hanya mengenal dua nilai logika: True dan False.

Tipe data lain bisa dianggap truthy (dipandang benar) atau falsy (dipandang salah) saat dinilai dalam kondisi.

Contoh Truthiness

  • 0, "", [], {}, None ➜ False
  • Selain itu ➜ True

Catatan: hari ini fokus ke boolean hasil perbandingan, tetapi penting tahu konsep ini sejak awal.

Tabel Kebenaran and

A B A and B
True True True
True False False
False True False
False False False

Operator and hanya bernilai True bila semua kondisi bernilai True.

Tabel Kebenaran or

A B A or B
True True True
True False True
False True True
False False False

Operator or bernilai True selama ada minimal satu kondisi True.

not & Kombinasi Kondisi

Membalik Nilai

Operator not membalik nilai boolean: not True ➜ False.

Cocok digunakan untuk memastikan sesuatu tidak terjadi.

Prioritas Evaluasi

  • not ➜ and ➜ or
  • Gunakan kurung () agar maksud jelas
  • Pahami urutan untuk mencegah bug logika

Contoh di Python

Kita gabungkan operator perbandingan dan logika untuk membuat keputusan. Perhatikan kurung untuk memisahkan kondisi. 

umur = 18
memiliki_sim = True
nilai_teori = 82

boleh_mengemudi = (umur >= 17 and memiliki_sim) and nilai_teori >= 75
izin_orangtua = umur < 21

if boleh_mengemudi and izin_orangtua:
    print("Silakan kendarai mobil dengan pendamping.")
else:
    print("Belum memenuhi semua syarat.")

Studi Kasus: Akses Kelas Premium

Skenario

Pengguna hanya boleh mengakses kelas premium jika sudah membayar atau mendapat kupon tutor, dan emailnya sudah terverifikasi.

  • Tentukan kondisi yang harus True atau False
  • Gunakan and dan or sesuai kebutuhan
  • Beri variabel boolean agar kode mudah dibaca
sudah_bayar = False
punya_kupon = True
email_terverifikasi = True

bisa_masuk = email_terverifikasi and (sudah_bayar or punya_kupon)

print("Akses diberikan?", bisa_masuk)

Latihan Mandiri

Buat program untuk menentukan apakah seorang siswa lulus kelas. Gunakan syarat:

  • Nilai akhir minimal 75
  • Kehadiran minimal 85%
  • Khusus siswa yang nilainya < 80 harus mengikuti remedial

Modifikasi logika sehingga program mencetak pesan berbeda untuk "Lulus Langsung", "Lulus Dengan Remedial", atau "Belum Lulus".

Ringkasan

  • Operator perbandingan menghasilkan nilai True atau False
  • and, or, not membantu menggabungkan logika kompleks
  • Gunakan variabel boolean & kurung untuk menjaga kode mudah dibaca
  • Latihan hari ini menjadi dasar membuat percabangan if di pertemuan berikutnya
