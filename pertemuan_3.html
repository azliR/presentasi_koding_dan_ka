Apa itu Python?

Python adalah bahasa pemrograman tingkat tinggi yang serbaguna, mudah dibaca, dan banyak digunakan di berbagai industri.

Sintaks Sederhana: Kodenya mirip bahasa Inggris, mudah dipelajari pemula.
Serbaguna (Versatile): Untuk web, AI, analisis data, otomasi, dan lainnya.
Komunitas Besar: Didukung komunitas global yang aktif, mudah mencari bantuan.

Relevansi Python di Industri

  • Pengembangan Web: Backend situs dengan Django & Flask.
  • Ilmu Data & AI: Andalan untuk analisis data dan sistem rekomendasi.
  • Otomasi & Skrip: Mengotomatiskan tugas komputer berulang.
  • Pengembangan Game: Membuat kerangka dasar game sederhana.

Opsi Lanjutan: Instalasi Lokal

Untuk penggunaan jangka panjang, Anda bisa menginstal Python langsung di komputer.

  1. Kunjungi python.org dan unduh versi terbaru.
  2. Saat instalasi, centang kotak "Add Python to PATH". Ini sangat penting!

Editor Kode: Anda butuh editor seperti VS Code, Sublime Text, atau PyCharm.

Keuntungan: Bisa coding kapan saja tanpa perlu koneksi internet.

Praktik 1: Menampilkan Teks

Fungsi print() adalah instruksi dasar untuk menampilkan teks atau informasi lain ke layar.

Contoh Kode:

print("Hello, World!")
print("Saya siswa SMK, siap belajar Python!")

Praktik 2: Menyimpan Informasi

Variabel bisa diibaratkan seperti sebuah kotak berlabel untuk menyimpan nilai atau informasi.

Contoh Kode:

nama = "Budi Hartono"
umur = "17"
sekolah = "SMK Negeri 1 Maju Jaya"

print(nama)
print(umur)
print(sekolah)

Praktik 3: Program Interaktif

Fungsi input() digunakan untuk menjeda program dan meminta masukan dari pengguna.

Contoh Kode:

print("Siapakah nama Anda?")
nama_pengguna = input()
print("Selamat datang, " + nama_pengguna + "!")

Studi Kasus: Program Biodata

Mari gabungkan semua konsep untuk membuat program yang lebih utuh. 

print("--- PROGRAM INPUT BIODATA ---")
nama = input("Masukkan Nama Lengkap: ")
kelas = input("Masukkan Kelas: ")
jurusan = input("Masukkan Jurusan: ")
umur = input("Masukkan Umur: ")

print("\n--- BIODATA ANDA ---")
print("Nama :", nama)
print("Kelas :", kelas)
print("Jurusan :", jurusan)
print("Umur :", umur, "tahun")

Latihan 1: Pembuat Slogan

Tugas: Buat program yang meminta nama produk dan kata sifat, lalu menggabungkannya menjadi slogan.

Petunjuk:

  • Minta input nama produk (misal: "Cokelat").
  • Minta input kata sifat (misal: "Lezat").
  • Gabungkan keduanya dan tampilkan hasilnya.

Contoh output:

Slogan baru Anda: Cokelat, Pasti Lezat!

Latihan 2: Generator Cerita

Tugas: Buat program "Mad Libs" yang meminta beberapa kata untuk digabungkan menjadi cerita lucu.

Petunjuk:

  • Minta input: warna, nama hewan, kata kerja, nama teman.
  • Simpan setiap kata dalam variabel berbeda.
  • Buat template cerita dan gabungkan dengan variabel.

Contoh output:

Kemarin, saya melihat seekor [nama hewan] berwarna [warna] sedang [kata kerja] bersama [nama teman].

Latihan 3: Kartu Nama Digital

Tugas: Buat program yang mencetak kartu nama sederhana berdasarkan input pengguna.

Petunjuk:

  • Minta input: Nama, Jabatan, Nomor Telepon.
  • Tampilkan hasilnya dalam format kartu nama yang rapi.

Contoh output:

=========================
Nama   : Budi Hartono
Jabatan: Siswa RPL
Kontak : 08123456789
=========================

Rangkuman Singkat

  • Kita bisa menampilkan output ke layar menggunakan print().
  • Kita bisa menyimpan data dalam variabel.
  • Kita bisa menerima masukan dari pengguna menggunakan input().
