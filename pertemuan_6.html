Percabangan If–Else di Python

  • Belajar membuat program yang mampu mengambil keputusan sederhana.
  • Fokus pada struktur if, else, dan elif.
  • Tujuan: memahami konsep percabangan dan membuat contoh programnya.

Program yang Bisa “Mikir”

Percabangan membuat program bertindak seperti manusia saat mengambil keputusan.

  • Jika hujan → pakai jas hujan.
  • Jika lapar → cari makanan.
  • Jika uang cukup → beli jajan.
  • Program pun bisa memilih jalur berdasarkan kondisi.
  • Itulah inti dari percabangan.

Apa Itu Percabangan?

  • Mengecek suatu kondisi.
  • Memilih jalur program berdasarkan hasil cek.
  • Jika kondisi benar → jalur A.
  • Jika salah → jalur lain (contohnya else).
  • Tanpa percabangan, kode hanya berjalan lurus.

Struktur Dasar if

Ingat dua aturan penting: titik dua dan indentasi. 

if kondisi:
    aksi_yang_dijalankan
  • Ada tanda : setelah kondisi.
  • Baris di dalam blok harus menjorok ke dalam.
  • Blok dijalankan hanya saat kondisi bernilai benar.

Contoh: Cek Kelulusan 

nilai = 80

if nilai >= 75:
    print("Lulus")
  • nilai berisi 80.
  • Kondisi nilai >= 75 bernilai benar.
  • Program mencetak “Lulus”.
  • Jika nilai < 75, tidak ada output.

Menambahkan else

Gunakan else ketika butuh aksi cadangan saat kondisi tidak terpenuhi. 

if kondisi:
    aksi_jika_benar
else:
    aksi_jika_salah

Hanya salah satu blok yang dieksekusi setiap kali percabangan berjalan.

Contoh: Lulus atau Tidak 

nilai = 60

if nilai >= 75:
    print("Lulus")
else:
    print("Tidak lulus")
  • Nilai 60 tidak memenuhi syarat.
  • Program melewati blok if dan menjalankan else.
  • Output: “Tidak lulus”.

Indentasi itu Penting

Di Python, indentasi menggantikan { } untuk menentukan blok kode.

Contoh Benar 

umur = 15

if umur >= 13:
    print("Remaja")
else:
    print("Anak-anak")

Contoh Salah 

umur = 15

if umur >= 13:
print("Remaja")
    else:
        print("Anak-anak")

Percabangan Bertingkat

Gunakan elif ketika lebih dari dua pilihan kondisi. 

if kondisi1:
    aksi1
elif kondisi2:
    aksi2
else:
    aksi_lain

Program mengecek dari atas ke bawah dan berhenti saat menemukan kondisi yang benar.

Contoh: Grade Nilai

nilai = 82

if nilai >= 90:
    print("Grade A")
elif nilai >= 80:
    print("Grade B")
elif nilai >= 70:
    print("Grade C")
elif nilai >= 60:
    print("Grade D")
else:
    print("Grade E")

Nilai 82 memenuhi syarat kedua sehingga langsung mencetak “Grade B”.

Rentang Nilai

  • 90–100 → Grade A
  • 80–89 → Grade B
  • 70–79 → Grade C
  • 60–69 → Grade D
  • < 60 → Grade E

Memakai Input Pengguna

  • input() meminta data dari pengguna.
  • Gunakan int() bila data harus berupa angka. 
nilai = int(input("Masukkan nilai kamu: "))

if nilai >= 75:
    print("Lulus")
else:
    print("Tidak lulus")

Program Ganjil / Genap 

angka = int(input("Masukkan sebuah angka: "))

if angka % 2 == 0:
    print("Angka genap")
else:
    print("Angka ganjil")
  • Operator % menghasilkan sisa bagi.
  • Sisa 0 berarti angka genap.
  • Selain itu dianggap ganjil.

Studi Kasus: Login

username = input("Masukkan username: ")
password = input("Masukkan password: ")

if username == "admin" and password == "1234":
    print("Login berhasil")
else:
    print("Username atau password salah")

Program hanya menerima pasangan data tertentu, mirip proses login aplikasi.

Studi Kasus: Diskon

total = int(input("Total belanja: "))

if total >= 100_000:
    diskon = total * 10 / 100
    total_bayar = total - diskon
    print("Mendapat diskon 10%")
    print("Total yang harus dibayar:", total_bayar)
else:
    print("Tidak ada diskon")
    print("Total yang harus dibayar:", total)

Diskon diberikan hanya ketika total belanja mencapai batas minimal.

Studi Kasus: Biaya Parkir

  • 2 jam pertama: Rp5.000.
  • Jam berikutnya: tambah Rp2.000 per jam.
jam = int(input("Lama parkir (jam): "))

if jam <= 2:
    biaya = 5_000
else:
    biaya = 5_000 + (jam - 2) * 2_000

print("Biaya parkir:", biaya)

Latihan Mandiri

1. Kategori Umur

  • Input umur dari pengguna.
  • < 13 → "Anak-anak".
  • 13–17 → "Remaja".
  • >= 18 → "Dewasa".
  • Gunakan if–elif–else.

2. Salam Berdasarkan Jam

  • Input jam (0–23).
  • 0–11 → "Selamat pagi".
  • 12–17 → "Selamat siang".
  • 18–23 → "Selamat malam".

Kesalahan Umum

  • Lupa menulis : setelah if, elif, atau else.
  • Indentasi tidak konsisten.
  • Salah pakai = ketika seharusnya ==.
  • Baris berada di luar blok karena jarak spasi salah.
  • Lupa mengubah input menjadi angka saat membandingkan numerik.

Rangkuman If–Else

Percabangan membuat program terasa hidup: memilih tindakan sesuai kondisi.

  • if mengecek kondisi.
  • else menangani kondisi yang tidak terpenuhi.
  • elif dipakai ketika pilihan lebih dari dua.
  • Indentasi menentukan blok kode.
  • Contoh penggunaan: login, diskon, tarif, kategori umur.
